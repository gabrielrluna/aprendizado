O que é HTML?

HTML (Hypertext Markup Language) é uma linguagem de marcação de texto. Permite que se crie estruturas textuais como parágrafos, seções, cabeçalhos, entre outros.

Título H1

Usando a tag "H1", você cria um título dentro do arquivo HTML. Para criar um parágrafo como este, basta usar a tag "p"

Título H2

Usando "H2", você cria um título de nível 2. Melhor dizendo, você cria um subtítulo.

Título H2

Usando "H3", você cria um título de nível 3. Quanto maior o número, menor o tamanho do título. O nível do "Hx" pode ir até 6.

Usando a tag "hr", você cria um separado como o que está acima deste parágrafo.

No HTML você pode fazer várias alterações no texto (apesar de não ser recomendado usar por aqui, mas num arquivo CSS)

Usando a tag "br" você deixa o texto em negrito (Também serve com "b")

Usando a tag "em" você deixa o texto em itálico (Também serve com "i")

Usando a tag "u" você deixa o texto sublinhado

Usando a tag "strike" você deixa o texto sublinhado
